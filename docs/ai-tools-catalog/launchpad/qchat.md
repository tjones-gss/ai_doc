# qchat

> **TL;DR** — Schedule quarterly check-ins. A lightweight tool to make sure the every-quarter career / status conversation actually happens, without manual calendar wrangling.

## Overview

Most people mean to do quarterly check-ins. They forget. qchat is a small scheduling helper that prompts both sides when the next check-in is due, lets you pick a time, and (optionally) drops a calendar invite.

## Why use it

- **The check-in actually happens.** A nudge beats good intentions.
- **Less calendar Tetris.** Pick from suggested times instead of mailing back-and-forth.

## When to use it

- You manage someone (or are managed) and want quarterly check-ins on autopilot.
- You're rolling out a new manager process and want consistency.

## How to access it

- **URL:** [https://launchpad.globalshopsolutions.dev/apps/qchat](https://launchpad.globalshopsolutions.dev/apps/qchat) → **Launch**.
- **Login:** Global Shop Office 365 SSO.
- **Status:** Live.

## How to use it

1. Launch the app.
2. Set up your quarterly cadence (or use the default).
3. When a check-in is coming up, pick a time from the suggestions.
4. Confirm — calendar invite is sent.
